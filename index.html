.addDate(duration, date)

Returns a new date that is the sum of date and duration

.diffDateDays(date1, date2)

Returns the difference between date1 and date2

.floor(token, date)

Returns a new date that is floored to the value given by token, where token is a string of value:

  • "mon" - "sun"
  • "day"
  • "week"
  • "fortnight"
  • "month"
  • "year"

.formatDate(template, timezone, locale, date)

Format date to template in the given timezone, where timezone is "UTC", "local" or an IANA timezone in the form "Europe/Zurich".

.formatTime(template, time)
