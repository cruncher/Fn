Fn 1.2.0

A library of functions, functors and streams.

core

cache(fn)

Returns a function that caches output values against input values.

const fn = cache(function(value) { console.log('run!'); return !!value; }); fn(1); // Logs 'run!', returns true fn(1); // Returns true fn(2); // Logs 'run!', returns true

compose(fn2, fn1)

Composes two functions into a single function, where fn2 is passed the result of fn1 and the result is returned.

var fn = compose(console.log, get(0)); fn(['Zero', 'One', 'Two']); // Logs 'Zero'

curry(fn)

var fn2 = curry(fn);

Curries fn. If fn normally requires 3 parameters, the curried result can take those parameters in any grouping:

fn2(a, b, c); fn2(a)(b)(c); fn2(a, b)(c);

By default a curried function expects immutable objects to be passed in. If a function expects only primitives there's no problem, but if it is to operate on objects that may change shape – or if it is a function with side effects – switch on mutability by passing true as the second parameter.

const fn2 = curry(fn1, true);

Lastly, curry reads a function's arity from fn1.length. Set the arity explicitly with a third parameter:

const fn2 = curry(fn1, true, 3); // fn2 expects 3 parameters

choose(object)

Returns a function that calls the function at property of object that matches it's first argument. It is called with the remaining arguments.

var fn = choose({ 'fish': function fn1(a, b) {...}, 'chips': function fn2(a, b) {...} }); fn('fish', a, b); // Calls fn1(a, b)

Accepts Map objects as well as plain objects.

overload(fn, object)

Returns a function that calls a function at the property of object that matches the result of calling fn (with all arguments).

var fn = overload(toType, { string: function a() {...}, number: function b() {...} }); fn('pie', 4); // Calls a('pie', 4) fn(1, 2); // Calls b(1, 2)

pipe(fn1, fn2, ...)

Composes functions into a pipe. fn2 is passed the result of fn1, fn3 is passed the result of fn2 and so on until the result of the last function is returned.

objects

assign(source, object)

Copies keys of object to source.

const a = {}; const fn = assign(a); fn({ haggis: 1 }); fn({ chips: 3 }); console.log(a); // { haggis: 1, chips: 3 }

get(key, object)

Gets property key of object, where object has a get method (eg. Map, WeakMap) or where key is a property of object.

const array = [{ word: 'steak' }, { word: 'pie' }]; array.map(get('word')); // ['steak', 'pie']

set(key, object, value)

Sets property key of object, where object has a set method (eg. Map, WeakMap) or where object can have key set on it.

const object = {}; ['steak', 'pie'].map(set('word', object)); console.log(object); // { word: 'pie' }

getPath(path, object)

Gets value at path in object, where path is a js notation path to a property (such as 'path[3].to.property').

const array = [{ object: { word: 'steak' }}, { object: { word: 'pie' }}]; array.map(getPath('object.word')); // ['steak', 'pie']

setPath(path, object, value)

Sets value at path in object, where path is a js notation path to a property (such as 'path[3].to.property').

const object = {}; ['steak', 'pie'].map(set('word', object)); console.log(object); // { word: 'pie' }

invoke(name, args, object)

Invokes method name of object with args.

const nodes = [node1, node2]; array.map(invoke('querySelectorAll', ['a'])); // [[<a> tags in node1], [<a> tags in node2]]

requestTick(fn)

Calls fn at the end of the current tick. (This helper is often called setImmediate in other libraries.)

throttle(fn, time)

Returns a function that calls fn periodically after it is called, with the latest context and arguments. By default calls are throttled to animation frames. Pass in time (in seconds) to use a setTimeout based timer, or pass in a custom timer object with a .request() function.

const fn = throttle(console.log, 1.5); fn(1); // Logs 1 fn(2); // ... fn(3); // Logs 3 after 1.5 seconds

wait(fn, time)

Returns a function that waits for time seconds without being invoked before calling fn using the context and arguments from its latest invocation.

const fn = wait(console.log, 1.5); fn(1); // ... fn(2); // ... fn(3); // Logs 3 after 1.5 seconds
